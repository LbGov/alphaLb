Colours

We use colours and graphics, including images and logos, in ways that support our website's purpose and ensures its accessibility.

We use limited colours to add emphasis to the text or content elements they're applied to.

Colour swatches

Blue and orange are our primary colours, so we use them first and only introduce other colours if they’re needed. Infographics or illustrations use these colours as a starting point.

Accessibility WebAim colour contrast checker

When we apply colour, we take into account how people might be using the website. For example, they may:

  • have a visual impairment
  • be viewing the site in bright sunlight
  • have printed a page in black and white.

Colour that has meaning

If colour has a meaning assigned to it, we use other methods to supplement this information. For example, a graph using colours also has text labels. For colour that has no meaning, like decorative text, or logos and brand names, we don't apply accessibility guidelines.

Contrast

We aim for good contrast between text and the background it appears on. To be effective, the contrast ratio between text colour and the background needs to be at least 4.5:1. For large text — 18 points/24 pixels upwards or, for bold text, 14 points/19 pixels upwards — we use a contrast ratio of 3:1.

We test the contrast ratio of our colours using an online tool.
